第一部分：一级结论

必背公式与定律
1一、力学

1. 运动学公式

(1) 匀变速直线运动

  • 速度公式：$v_t = v_0 + at$
  • 位移公式：$s = v_0t + \frac{1}{2}at^2$
  • 速度位移公式：$v_t^2 - v_0^2 = 2as$
  • 平均速度（定义式）：$\bar{v} = \frac{s}{t}$

(2) 三个重要推论

  • 中间时刻速度：$v_{\frac{t}{2}} = \bar{v} = \frac{v_0 + v_t}{2}$
  • 中间位置速度：$v_{\frac{s}{2}} = \sqrt{\frac{v_0^2 + v_t^2}{2}}$
  • 位移差公式：$\Delta s = aT^2$（相邻连续相等时间内的位移之差）

(3) 自由落体运动（$v_0=0,\ a=g$）

  • $v_t = gt$，$h = \frac{1}{2}gt^2$，$v_t^2 = 2gh$

(4) 竖直上抛运动

  • $s = v_0t - \frac{1}{2}gt^2$，$v_t = v_0 - gt$
  • 最大高度：$H_m = \frac{v_0^2}{2g}$，总时间：$t = \frac{2v_0}{g}$

(5) 平抛运动

  • 水平：$v_x = v_0$，$s_x = v_0t$
  • 竖直：$v_y = gt$，$s_y = \frac{1}{2}gt^2$
  • 合速度：$v_t = \sqrt{v_0^2+(gt)^2}$
  • 运动时间：$t = \sqrt{\frac{2h}{g}}$（由下落高度决定）

(6) 匀速圆周运动

  • $v = \omega r$，$a_n = \frac{v^2}{r} = \omega^2 r$
  • $F_n = m\frac{v^2}{r} = m\omega^2 r$
  • $T = \frac{1}{f} = \frac{2\pi r}{v} = \frac{2\pi}{\omega}$

2. 力的相互作用

  • 重力：$G=mg$（$g\approx 9.8\text{ m/s}^2$）
  • 胡克定律：$F=kx$
  • 滑动摩擦力：$f=\mu F_N$
  • 静摩擦力：$0\leq f\leq f_{\max}$
  • 合力范围：$|F_1-F_2|\leq F\leq F_1+F_2$
  • 正交分解：$F_x=F\cos\theta,\ F_y=F\sin\theta$

3. 牛顿运动定律

  • 牛顿第二定律：$F_{\text{合}}=ma$
  • 牛顿第三定律：$F_{AB}=-F_{BA}$

4. 万有引力与天体运动

  • $F=G\frac{m_1 m_2}{r^2}$，$G=6.67\times10^{-11}\text{ N·m}^2/\text{kg}^2$
  • 黄金代换：$GM=gR^2$
  • 卫星：$v=\sqrt{\frac{GM}{r}}$，$T=2\pi\sqrt{\frac{r^3}{GM}}$
  • 第一宇宙速度：$v_1=\sqrt{gR}\approx7.9\text{ km/s}$
  • 第二宇宙速度：$v_2\approx11.2\text{ km/s}$
  • 第三宇宙速度：$v_3\approx16.7\text{ km/s}$

5. 功与能

  • 功：$W=Fs\cos\theta$；功率：$P=\frac{W}{t}=Fv$
  • 动能：$E_k=\frac{1}{2}mv^2$；重力势能：$E_p=mgh$
  • 弹性势能：$E_p=\frac{1}{2}kx^2$
  • 动能定理：$W_{\text{合}}=\frac{1}{2}mv_2^2-\frac{1}{2}mv_1^2$
  • 机械能守恒：$E_{k1}+E_{p1}=E_{k2}+E_{p2}$（仅有重力或弹力做功）

6. 动量

  • 动量：$p=mv$；冲量：$I=Ft$
  • 动量定理：$Ft=mv'-mv$
  • 动量守恒：$m_1v_1+m_2v_2=m_1v_1'+m_2v_2'$（合外力为零）

弹性碰撞：

$$v_1' = \frac{m_1-m_2}{m_1+m_2}v_1 + \frac{2m_2}{m_1+m_2}v_2$$ $$v_2' = \frac{2m_1}{m_1+m_2}v_1 + \frac{m_2-m_1}{m_1+m_2}v_2$$

完全非弹性碰撞：$v'=\frac{m_1v_1+m_2v_2}{m_1+m_2}$（动能损失最大）

7. 机械振动与机械波

  • 简谐运动回复力：$F=-kx$
  • 弹簧振子周期：$T=2\pi\sqrt{\frac{m}{k}}$
  • 单摆周期：$T=2\pi\sqrt{\frac{L}{g}}$
  • 波速：$v=\lambda f=\frac{\lambda}{T}$
2二、热学

1. 分子动理论

  • 阿伏加德罗常数：$N_A=6.02\times10^{23}\text{ mol}^{-1}$
  • 分子直径数量级：$10^{-10}\text{ m}$

2. 热力学定律

  • 热力学第一定律：$\Delta U=W+Q$
  • 热力学第二定律（克劳修斯表述）：热量不能自发地从低温物体传到高温物体

3. 理想气体状态方程

  • $pV=nRT$，$\frac{p_1V_1}{T_1}=\frac{p_2V_2}{T_2}$
  • 等温：$p_1V_1=p_2V_2$（玻意耳定律）
  • 等压：$\frac{V_1}{T_1}=\frac{V_2}{T_2}$（盖-吕萨克定律）
  • 等容：$\frac{p_1}{T_1}=\frac{p_2}{T_2}$（查理定律）

4. 热量计算

  • $Q=cm\Delta t$，热值$q=\frac{Q}{m}$，热机效率$\eta=\frac{Q_{\text{有效}}}{Q_{\text{总}}}$
3三、电磁学

1. 静电场

  • 库仑定律：$F=k\frac{q_1 q_2}{r^2}$，$k=9.0\times10^9\text{ N·m}^2/\text{C}^2$
  • 电场强度：$E=\frac{F}{q}$（定义式）；点电荷：$E=k\frac{Q}{r^2}$；匀强：$E=\frac{U}{d}$
  • 电势：$\varphi=\frac{E_p}{q}$；电势差：$U_{AB}=\varphi_A-\varphi_B$
  • 电场力做功：$W_{AB}=qU_{AB}$
  • 电容：$C=\frac{Q}{U}$；平行板：$C=\frac{\varepsilon_r S}{4\pi k d}$

2. 恒定电流

  • 欧姆定律：$I=\frac{U}{R}$；电阻定律：$R=\rho\frac{L}{S}$
  • 电功率：$P=UI=I^2R=\frac{U^2}{R}$
  • 闭合电路欧姆定律：$I=\frac{E}{R+r}$，$U_{\text{路端}}=E-Ir$
  • 串联：$I$相同，$U_{\text{总}}=\sum U_i$，$R_{\text{总}}=\sum R_i$
  • 并联：$U$相同，$I_{\text{总}}=\sum I_i$，$\frac{1}{R_{\text{总}}}=\sum\frac{1}{R_i}$

3. 磁场

  • $B=\frac{F}{IL}$（$I\perp B$）；安培力：$F=BIL$
  • 洛伦兹力：$f=qvB$（$v\perp B$），左手定则判方向
  • 带电粒子圆周运动：$r=\frac{mv}{qB}$，$T=\frac{2\pi m}{qB}$

4. 电磁感应

  • 法拉第定律：$E=n\frac{\Delta\Phi}{\Delta t}$
  • 切割磁感线：$E=BLv$（$B,L,v$两两垂直）
  • 磁通量：$\Phi=BS\cos\theta$
  • 自感电动势：$E=L\frac{\Delta I}{\Delta t}$

5. 交变电流

  • $e=E_m\sin\omega t$，$E_m=nBS\omega$
  • 有效值：$E_{\text{有}}=\frac{E_m}{\sqrt{2}}$，$U_{\text{有}}=\frac{U_m}{\sqrt{2}}$，$I_{\text{有}}=\frac{I_m}{\sqrt{2}}$
  • 理想变压器：$\frac{U_1}{U_2}=\frac{n_1}{n_2}$，$I_1U_1=I_2U_2$
4四、光学
  • 折射定律：$n_1\sin\theta_1=n_2\sin\theta_2$
  • 绝对折射率：$n=\frac{c}{v}=\frac{\sin\theta_1}{\sin\theta_2}$
  • 全反射临界角：$\sin C=\frac{1}{n}$
  • 双缝干涉条纹间距：$\Delta x=\frac{L}{d}\lambda$
  • 亮条纹：$\delta=k\lambda$（$k=0,1,2,\ldots$）
  • 暗条纹：$\delta=(2k+1)\frac{\lambda}{2}$
  • 光子能量：$E=h\nu$（$h=6.63\times10^{-34}\text{ J·s}$）
  • 光电效应方程：$E_k=h\nu-W_0$
  • 德布罗意波长：$\lambda=\frac{h}{p}$
5五、原子物理
  • 玻尔氢原子：$E_n=\frac{E_1}{n^2}$（$E_1=-13.6\text{ eV}$），$r_n=n^2r_1$（$r_1=0.053\text{ nm}$）
  • 能级跃迁：$h\nu=E_m-E_n$
  • 质能方程：$\Delta E=\Delta m\cdot c^2$
  • 核反应守恒：质量数守恒、电荷数守恒
  • 半衰期：$m=m_0\!\left(\frac{1}{2}\right)^{\frac{t}{T_{1/2}}}$
Ⅱ

第二部分：二级结论

重要推论速查
1-4一、运动学
  1. 匀变速直线运动中：$V_{\frac{s}{2}} > V_{\frac{t}{2}}$（中间位置速度始终大于中间时刻速度）
  2. 初速度为零的匀加速直线运动比例式：
    • 速度比：$v_1:v_2:v_3:\cdots=1:2:3:\cdots$
    • 前$nT$位移比：$s_1:s_2:s_3:\cdots=1^2:2^2:3^2:\cdots$
    • 第$n$个$T$内位移比：$1:3:5:\cdots:(2n-1)$
    • 通过相等位移时间比：$1:(\sqrt{2}-1):(\sqrt{3}-\sqrt{2}):\cdots$
  3. 平抛运动推论：$\tan\theta_{\text{速}}=2\tan\theta_{\text{位}}$；速度反向延长线交于水平位移中点
  4. 竖直上抛对称性：上升与下落同一点速度大小相等、方向相反
5-9二、静力学
  1. 合力范围：$|F_1-F_2|\leq F\leq F_1+F_2$
  2. 三力平衡：三个大小相等、互成120°的力合力为零；任意两力合力与第三力等大反向
  3. 绳的张力方向：沿绳指向绳收缩的方向
  4. 摩擦角与自锁：当$\theta\leq\arctan\mu$时物体自锁（斜面不下滑）
  5. 力的相似三角形法：三力平衡时力矢量三角形与几何三角形相似
10-12三、动力学
  1. 等时圆结论：从竖直圆最高点沿弦滑到圆周任意点时间与直径自由落体相等；45°弦时间最短
  2. 整体法与隔离法：静止或匀速时$N=(M+m)g,\ f=0$；水平面光滑有加速度时$N<(M+m)g,\ f=ma$
  3. 弹簧双振子：两物体中间夹弹簧同时释放，动量、机械能均守恒
13-15四、万有引力
  1. 人造卫星性质：半径越大，$v$越小、$\omega$越小、$T$越大、$a$越小
  2. 同步卫星：$T=24\text{ h}$，赤道平面上，高度约$3.6\times10^4\text{ km}$
  3. 双星系统：$m_1\omega^2 r_1=m_2\omega^2 r_2$，$r_1+r_2=L$，$T=2\pi\sqrt{\frac{L^3}{G(m_1+m_2)}}$
16-18五、机械振动与波
  1. 简谐运动对称性：靠近平衡位置时$a$减小而$v$增大；远离时$a$增大而$v$减小
  2. 波的干涉：加强点$|\Delta s|=k\lambda$；减弱点$|\Delta s|=(2k+1)\frac{\lambda}{2}$
  3. 波形平移法：沿传播方向平移$x=vt$得新波形
19-21六、功与能
  1. 变力做功：可用$F$-$s$图像面积求解
  2. 机车启动：恒功率启动$a$减$v$增，最终$v_m=\frac{P}{f}$；恒加速度启动匀加速末$F>f$后切换恒功率
  3. 弹簧系统：两物体与弹簧，动量守恒+机械能守恒；最短或最长时两物体速度相等
22-23七、动量
  1. 弹性碰撞特殊情况：
    • $m_1=m_2$时：速度交换
    • $m_1\gg m_2$时：$v_1'\approx v_1$，$v_2'\approx2v_1$
    • 动球碰静球：$v_1'=\frac{m_1-m_2}{m_1+m_2}v_1$，$v_2'=\frac{2m_1}{m_1+m_2}v_1$
  2. 人船模型：$M l_m=-m L_M$（位移与质量成反比）
24-31八、电学
  1. 并联电路：总电阻小于任一支路；支路电阻增大则总电阻增大
  2. 滑动变阻器分压式：输出电压$0\to U$连续可调
  3. 电源输出功率：当$R_{\text{外}}=r$时最大，$P_{\max}=\frac{E^2}{4r}$
  4. 带电粒子匀强电场偏转：$y=\frac{qUL_0^2}{2mdv_0^2}$，$\tan\theta=\frac{qUL_0}{mdv_0^2}$
  5. 电势能变化：$W_{\text{电}}=-\Delta E_p$
  6. 圆形磁场：径向射入必径向射出
  7. 楞次定律推论："来拒去留"、"增反减同"
  8. 理想变压器决定关系：副线圈电压由原线圈决定；原线圈电流和功率由副线圈决定
32-34九、热学
  1. 宏观量与微观量联系：通过阿伏加德罗常数$N_A$联系
  2. 判断理想气体过程：①状态参量+气体定律；②热力学第一定律+能量角度
  3. 热力学第一定律符号法则：体积增大气体对外做功$W$为"$-$"；体积减小外界对气体做功$W$为"$+$"
35-38十、光学与原子物理
  1. 全反射条件：光密→光疏 + 入射角$\geq C$
  2. 可见光中：红光折射率最小、紫光最大；同一介质中红光速度最大
  3. 核能计算：$\Delta E=\Delta m\cdot c^2$，$1\text{ u}$质量亏损对应$931.5\text{ MeV}$
  4. 简谐运动（补充）：靠近平衡位置$a$减小$v$增大；远离平衡位置$a$增大$v$减小
